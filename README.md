# dev-faker-frontend
KT caravan dev-faker Frontend Project

### Vue3 개발환경 구성하기
#### 1. node 설치
```
$ node -v
v16.14.2
```
#### 2. Vue CLI 설치
```
$ npm install -g @vue/cli
(permission error가 발생하면 sudo로 실행)
```
아래 명령어로 vue 설치 확인
```
$ vue -V
@vue/cli 5.0.4
```
### 3. Vue 프로젝트 구성
```
$ vue create dev-faker

(Vue 버전을 선택하는 항목에서 반드시 Vue 3 선택!
Please pick a preset: Default ([Vue 3] babel, eslint)
```
